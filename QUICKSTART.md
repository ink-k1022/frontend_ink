# 🚀 快速啟動指南

## 立即開始使用

### 方法一：直接開啟（推薦）

1. 雙擊 `index.html` 文件
2. 在彈出的位置權限請求中點擊「允許」
3. 開始探索周邊店家！

### 方法二：本地伺服器（更佳體驗）

如果您安裝了 Python：

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

然後在瀏覽器訪問：`http://localhost:8000`

### 方法三：使用 VS Code Live Server

1. 安裝 "Live Server" 擴充功能
2. 右鍵點擊 `index.html`
3. 選擇 "Open with Live Server"

## ⚙️ 權重調整說明

### 距離權重 (Wd)
- **0%**：完全忽略距離，只看評價
- **50%**：平衡考慮距離和評價
- **100%**：只考慮距離，忽略評價

### 評價權重 (Wr)
- **0%**：完全忽略評價，只看距離
- **50%**：平衡考慮距離和評價
- **100%**：只考慮評價，忽略距離

💡 **提示**：兩個權重會自動互補，總和始終為 100%

## 📊 理解智能分數

**智能分數**是系統根據您設定的權重計算出的綜合推薦指數（0-100分）

### 分數構成
```
智能分數 = 距離權重 × 距離分數 + 評價權重 × (評價分數 + 評論數加成)
```

### 分數解讀
- **90-100分**：極度推薦 ⭐⭐⭐⭐⭐
- **80-89分**：強烈推薦 ⭐⭐⭐⭐
- **70-79分**：值得考慮 ⭐⭐⭐
- **60-69分**：可以參考 ⭐⭐
- **<60分**：不太推薦 ⭐

## ❓ 常見問題

### Q: 為什麼沒有顯示我的位置？
A: 請確認已允許瀏覽器的位置權限。如果仍無法定位，系統會使用預設位置（台北101）。

### Q: 為什麼找不到店家？
A: 
1. 檢查搜尋半徑是否太小
2. 降低最低評分要求
3. 取消「僅顯示營業中」篩選
4. 清空搜尋關鍵字和分類篩選

### Q: 如何重新定位？
A: 點擊地圖右上角的 🎯 定位按鈕即可。

### Q: 手機版如何查看結果列表？
A: 結果列表會自動顯示在畫面下方，可以向上滑動查看更多。

### Q: 可以離線使用嗎？
A: 目前版本需要網路連線以載入地圖圖層。未來版本將支援離線模式。
