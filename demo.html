📍 系統簡介

這是一個基於地理圍欄技術的智能搜尋引擎，結合距離與評價的多維權重演算法，幫您找到周邊最合適的店家。

✨ 核心特色

精準定位：自動感應位置，即時搜尋周邊店家
智能排序：動態平衡距離與評價
權重調整：自由調整優先級滿足不同需求
視覺化地圖：直觀顯示店家位置
多維篩選：分類、評分、營業狀態等多重條件

💡 使用提示

首次使用請允許位置權限，以獲得最佳體驗。系統會自動定位並顯示您周邊的優質店家。

支援三種模式：

  • 🚀 距離優先 - 適合緊急需求（加油站、超商）
  • ⚖️ 智能平衡 - 綜合考量距離與評價
  • 🏆 品質優先 - 找高評價的精緻服務

🎯 權重公式

Score = Wd × 距離分數 + Wr × (評價分數 + 評論數加成)

其中：

  • Wd = 距離權重（0-100%）
  • Wr = 評價權重（0-100%）
  • 距離分數：越近分數越高（歸一化 0-1）
  • 評價分數：星級評分轉換（0-1）
  • 評論數加成：最高 20% 額外加分

📊 技術架構

  • 🎨 前端: HTML5 + CSS3 + JavaScript (ES6+)
  • 🗺️ 地圖: Leaflet.js 1.9.4 + OpenStreetMap
  • 📍 定位: Geolocation API
  • 📐 演算法: Haversine 距離計算 + 權重評分系統
